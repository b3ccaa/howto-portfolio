Responsive Design

What does it mean?

It's a fact: We don't only use the internet on our computers anymore. Over the last few years different portable devices (e.g. smartphones or tablet devices) have been introduced to access websites. Due to that, the design and implementation of web pages needed to change to respond to the various screen sizes, resolutions and orientation of the handheld devices.

Responsive web design consists of flexible grids and layouts, adjusting images and making use of scripts and CSS media queries. Thereby the optimal solution is to dynamically adjust the website by showing or hiding content based on the screen size.
Images can be a challenging topic in responsive web design. You can resize images for smaller devices - even better replace them with smaller images, for instance with the help of a script. Especially on smartphones you don't want to have big images that don't look nice on small screens and use up a user's bandwidth cap for downloading them.

The simpler the design of a website is, the easier it is to make it responsive. There are some key roles to improve the mobile experience:

  • Simple navigation; no flyout menus
  • Focus on content
  • Use lists and rows instead of columns

Because of the big amount of various devices, it is hard to test websites regarding their responsiveness on all of them to make sure that your HTML and CSS is correct. However, testing is important.
