Welcome to HowTo Portfolio

This project has been done to help you setting up your own portfolio. All basic HTML and CSS have been implemented, and are ready to be extended and customized by changing colors, styles, font types or images.

This basic portfolio has been created with respect on responsive webdesign and thus will look good on desktop and mobile devices.

In the menu you can find more information about the HowTo Portfolio and how you can set up your own one.

Enjoy creating your own portfolio and feel free to adjust this basic setup to make it more personal.
