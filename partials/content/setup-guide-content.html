Setup Guide

To make use of this portfolio project, you need to do the following steps to set it up.

  1. Install Node-js
    Node.js is a package manager. We it need to install Grunt (see step 2).

  2. Install Grunt
    We need Grunt to build complete HTML files from HTML partials. More information can be found in step 5.

  3. Install a Less compiler, e.g. for Windows: WinLess or for Mac SimpLESS
    We use the CSS pre-processor Less, to extend the CSS language by adding features that allow variables, mixins, functions and many other techniques. Less also helps you maintain your CSS better. If you are new to Less, check out this website: lesscss.org

  4. Download the HowTo Responsive Portfolio Project here and unzip it. This is the base for your own responsive portfolio.

  5. Install Grunt package dependencies
    Go to the root of the just downloaded HowTo Responsive Portfolio Project (inside the folder "howto_portfolio"). Open a console/terminal from here and type in:
    npm install
    You can find more info about this step here: Working with an existing Grunt project.

  6. Install a good editor for developing your own portfolio (e.g. Sublime Text)
